Student Management System

A simple and efficient Student Management System developed using C programming language.

Project Description

The Student Management System is a console-based application developed using C programming. This project helps manage student records efficiently. Users can add, view, search, and delete student details using a simple menu-driven interface.

Features

  • Add Student Records
  • View All Students
  • Search Student by Roll Number
  • Delete Student Records
  • Simple Menu Driven Interface
  • Fast and Efficient Data Handling
